# Sales Data Analysis Dashboard

### Dashboard Link
https://app.powerbi.com/

---

## Problem Statement

This dashboard helps businesses analyze sales performance, profit trends, customer behavior, and product performance. It enables users to identify top-performing and low-performing products, track sales trends over time, and gain meaningful business insights through interactive visualizations.

The dashboard also helps businesses evaluate discount impact, compare performance between different periods, and analyze sales across different cities for better decision-making.

---

## Steps Followed

- Step 1 : Loaded the sales dataset into Power BI Desktop.
- Step 2 : Opened Power Query Editor for data cleaning and transformation.
- Step 3 : Checked column quality, distribution, and profiling.
- Step 4 : Removed null values and corrected data types.
- Step 5 : Created relationships using Primary Keys and Foreign Keys.
- Step 6 : Implemented Star Schema and Cardinality for efficient data modeling.
- Step 7 : Created DAX measures for Sales, Profit, Quantity Sold, and Discounts.
- Step 8 : Added slicers and visual filters for Product, Date, Customer ID, and Promotion Categories.
- Step 9 : Built KPI cards for Total Sales, Profit, Quantity Sold, and Orders.
- Step 10 : Created visualizations for:
  - Top/Bottom 5 Products by Sales, Profit, and Quantity
  - Sales Trends (Daily, Monthly, Quarterly, Yearly)
  - Sales vs Profit Analysis
  - City-wise Sales Analysis
  - Discount Category Analysis
- Step 11 : Added comparison visuals for comparing different periods.
- Step 12 : Designed an interactive and user-friendly dashboard layout.
- Step 13 : Published the dashboard to Power BI Service.
